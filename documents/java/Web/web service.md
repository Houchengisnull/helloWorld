# 定义

**webservice实质是http与xml的结合。**

而`SOAP`同样是`http`与`xml`的结合，但并非是`webservice`专有协议。

## WSDL

`WebService Description Language`，Web服务描述语言。

## SOAP

`SOAP-Simple Object Access Protocol`，简单对象访问协议。

- **组成**
  - **Envelope**	必须部分，`XML`的根元素
  - **Headers**	可选
  - **Body**	必须

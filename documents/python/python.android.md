# Weditor

## Usage

``` shell
# 安装
# 打开cmd窗口输入set PYTHONUTF8=1 启用python环境编码为utf-8模式
set PYTHONUTF8=1
pip install weditor

# 启动 weditor服务端口17130
python -m weditor

# 安装apk服务到手机
python -m uiautomator2 init
```

# UIAuto

## Overview

由于Weditor代码难以维护，其作者重新开发的产品。

- 实时刷新设备页面
- 可查看元素树

## Uasge

``` shell
# install
pip3 install -U uiautodev -i https://pypi.doubanio.com/simple

# 启动
uiauto.dev
python3 -m uiautodev
```
